# VIM插件——vimplus安装（centos 7）

Centos
版本要求
centos7及其以上系统。
安装vimplus(建议在普通用户下安装)

git clone https://github.com/chxuan/vimplus.git ~/.vimplus
cd ~/.vimplus
./install.sh

快捷键
以下是部分快捷键，更详细的快捷键请查阅vimplus帮助文档

<F5>	显示语法错误提示窗口
<F7>	启用markdown实时预览
<F8>	关闭markdown实时预览
<F9>	显示上一主题
<F10>	显示下一主题
Ya		复制行文本到字母a
Da		剪切行文本到字母a
Ca		改写行文本到字母a
rr			替换文本
<leader>r		全局替换，目前只支持单个文件
gcc			注释代码
gcap		注释段落
