## 转义

| **转义序列**       | **含义**                 | **整数编码** |
| ------------------ | ------------------------ | ------------ |
| \b                 | 退格符                   | 8            |
| \d                 | 删除符                   | 127          |
| \e                 | 换码符                   | 27           |
| \f                 | 换页符                   | 12           |
| \n                 | 换行符                   | 10           |
| \r                 | 回车符                   | 13           |
| \s                 | 空格符                   | 32           |
| \t                 | 制表符                   | 9            |
| \v                 | 垂直制表符               | 11           |
| \x{...}            | 十六进制字符             |              |
| ^a...^z或者^A...^Z | Ctrl+a...Ctrl+Z          | 1~26         |
| '                  | 单引号                   | 39           |
| "                  | 双引号                   | 34           |
| \                  | 反斜杠                   | 92           |
| \C                 | C的ASCII编码(C是一个字符 | 一个整数     |
